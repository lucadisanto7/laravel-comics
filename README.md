Create un nuovo progetto Laravel.
Concentratevi sul layout: create un file di layout in cui inserire la struttura comune di tutte le pagine del sito web (tag head, tag body, ...) eventualmente includendo header e footer tramite due partials. Create poi una rotta per visualizzare la lista di tutti i fumetti recuperati da un file inserito nella cartella config. Non concentratevi sulla grafica, intanto strutturate il tutto come descritto sopra, utilizzate al limite le classi predefinite di bootstrap.
